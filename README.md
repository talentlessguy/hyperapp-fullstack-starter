## Install

```sh
git clone https://github.com/talentlessguy/hyperapp-fullstack-starter.git
cd hyperapp-fullstack-starter
pnpm install
pnpm dev
```

## Development

```sh
pnpm dev
```

## Production

```sh
pnpm build # build a production version
pnpm start # start the server
```
